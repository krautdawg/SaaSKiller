# Design Documentation - SaaSKiller

## Overview
This directory contains the comprehensive design system and feature specifications for the SaaSKiller platform.

## Directory Structure

- **design-system/**: Core visual language, tokens, and components.
  - `style-guide.md`: The central source of truth for branding, color, and typography.
  - `components/`: Detailed specs for UI elements.
  - `tokens/`: Atomic design values.
- **features/**: Feature-specific design briefs and user journeys.
  - `saas-audit/`: Documentation for the core "Hit List" and Audit flow.
- **accessibility/**: Guidelines and compliance check-lists.
- **assets/**: References and exported tokens.
